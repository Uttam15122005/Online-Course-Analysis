## 🎯 Overview
This project analyzes online course data to extract meaningful insights related to **pricing, subscribers, engagement, and course trends** using an interactive Power BI dashboard.

---

## ⚡ Key Features
✨ Interactive dashboards with dynamic filters  
📊 KPI tracking (Subscribers, Reviews, Price)  
📈 Subject-wise and year-wise analysis  
💰 Revenue and pricing insights  
🔍 Paid vs Free course comparison  

---

## 🛠️ Tech Stack
- 🐍 Python (Data Preprocessing)  
- 📊 Power BI (Dashboard Visualization)  
- 📑 Excel (Data Handling)
